# opoclaw
opoclaw is a OpenClaw alternative built in Bun made to be fast and sandboxed. It is entirely safe to run by default, simple and easy to use, and a polished experience.
|                       | **Opoclaw**          | OpenClaw   | NanoClaw   | PicoClaw |
|-----------------------|----------------------|------------|------------|----------|
| Language              | **Bun (TypeScript)** | TypeScript | TypeScript | Go       |
| RAM                   | **<32 MB**           | <1 GB      | <500 MB    | <10 MB   |
| Startup (0.8GHz core) | **<1s**              | <500s      | <30s       | <1s      |
